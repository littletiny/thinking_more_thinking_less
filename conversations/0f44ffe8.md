**关键洞察**：
- 高频小 Commit 策略：每个文件独立 commit，失败只回滚该文件
- Git 让"实验"成为可能：branch 上任意尝试，失败丢弃，成功合并
- Agent 专用工作流：
  ```bash
  for file in files_to_edit:
      edit(file)
      git add file
      git commit -m "agent: update ${file}"
      if verification_failed:
          git revert HEAD
  ```
