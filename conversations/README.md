## 格式说明

每个对话文件包含：

1. **Metadata**：日期、主题、参与者、产出概念
2. **完整对话记录**：
   - `### User Input` - 用户原文，完整复制
   - `### AI Output (Complete)` - AI完整回复，不摘要
3. **Key Quotes (Exact)**：关键原话摘录
4. **Concept Extraction**：从对话提炼的概念
5. **Follow-up Impact**：对后续设计的影响

---

## 记录原则

| 原则 | 说明 |
|------|------|
| **保留原文** | 不编辑、不摘要、不删减用户输入 |
| **保留完整AI输出** | 不只是结论，包含完整推理过程 |
| **保留年份2026** | 所有对话文件使用2026年 |
| **可追溯** | 每个概念必须链接到来源对话 |

## 使用方式

- **追溯起源**：查看概念是如何从具体讨论中产生的
- **理解决策**：了解为什么选择某种设计
- **学习过程**：不只是结论，学习思考的过程
