# 在 Obsidian 中使用本知识库

## 快速开始

### 1. 安装 Obsidian
下载: https://obsidian.md/

### 2. 打开知识库
```
1. 打开 Obsidian
2. 选择 "Open folder as vault"
3. 选择 zettel/ 目录
4. 完成！
```

### 3. 查看知识图谱
```
1. 点击左侧面板的 "Graph view" 图标
2. 你会看到所有概念的可视化网络
3. 点击任意节点查看详情
4. 拖动、缩放探索关联
```

## 图谱解读

### 节点颜色
- 蓝色: concepts/ 概念卡片
- 绿色: conversations/ 对话记录
- 紫色: connections/ 连接分析
- 灰色: notes/ 系统文档

### 节点大小
- 越大 = 被引用次数越多
- 核心概念会显示更大

### 连线
- 实线: 显式链接 ([[concept-name]])
- 粗细: 关联强度

## 实用技巧

### 1. 聚焦视图
```
在 Graph view 中:
- 选中一个节点
- 点击 "Local graph"
- 只看到相关概念
```

### 2. 过滤显示
```
Graph view 设置:
- 只显示 concepts/
- 隐藏对话记录
- 按标签过滤
```

### 3. 发现连接
```
在任意笔记中:
- 查看 "Backlinks" 面板
- 发现哪些概念引用了当前页
- 发现意外的关联
```

## 示例：探索 DIY 系统

```
1. 搜索 "diy-software-mindset"
2. 打开该笔记
3. 查看 Local graph
4. 你会看到:
   - 中心: diy-software-mindset
   - 周围: 
     - hard-constraints-system
     - mock-driven-validation
     - scaffold-not-platform
   - 连线显示它们的关系
```
