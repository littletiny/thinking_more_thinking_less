# Frequency-Value Paradox / 频次-价值悖论

## One Sentence
工具型产品面临的核心张力——高价值场景（重大决策）天然低频，而高频场景（日常选择）又缺乏咨询的边际价值，导致产品难以建立用户黏性。

---

## Why It Matters

这是决策辅助产品不可行的**结构性原因**：

| 场景 | 频次 | 价值 | 用户行为 |
|-----|------|------|---------|
| 重大决策 | 低 | 高 | 愿意咨询，但场景太少 |
| 日常决策 | 高 | 低 | 懒得咨询，自己做更快 |

**结果**: 产品要么没有存在感（低频），要么没有价值感（低价值）。

## Core Mechanism

```
用户黏性要求高频使用
     ↑
高价值场景天然低频
     ↑
低频高价值 ≠ 日常黏性
     ↓
产品死亡区
```

---

## Attempted Solutions (Failed)

1. **情感陪伴伪装** → 薅 token，情绪价值与决策价值分离
2. **累积决策质量** → 传销话术，用户不会为"复利"买单
3. **低频高价服务** → 渠道问题，获客成本决定生死
4. **决策基础设施** → 直面巨头竞争，无护城河
5. **B2B2C** → SaaS 被 agent 取代的风险

**结论**: 这不是执行问题，是方向本身的结构性矛盾。
